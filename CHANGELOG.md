# Changelog

## 0.1.0

Initial release.

- Sidebar dashboard with live TOML file watching
- Branch-aware automatic file loading (branch slashes become `--`)
- Task cards with expandable subtask details
- Circular progress rings (per-task and overall)
- Plan and Build prompt clipboard copy
- Scope drift tracking with visual tags
- Out-of-scope item tracking
- Burndown chart visualization
- Cursor Rule and CLAUDE.md content copy buttons
- /taskpilot-init, /taskpilot-sync, /taskpilot-add command content
- Privacy-first design -- all data stays local
- Auto-gitignore for `.taskpilot/` directory
